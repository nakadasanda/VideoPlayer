# VideoPlayer 　　

windows向けビデオプレイヤー(映像のみ)  

これはQtプロジェクトです。  
QtCreaterで開くことができます。  
  
  
FFMPEGのバージョンは、4.3です。  
Qtのバージョンは、5.15です。  
c++のバージョンは、c++11です。  

____________
## ブランチの分け方
[0.0.0]ffnpeg,qtの導入  
[0.1.0]ffmpegで動画ファイル(mp4)から画像ファイル(ppm)の抽出  
[0.2.0]ffmpegで動画ファイルの映像再生。
